Description


تعلم تصميم المواقع بشكل مجاني كامل حيث سنوفر عليك سنوات من التعب فسننقل لك خبرة أعوام في أيام فتعلم دائما الافضل لأنك ستتعلم في هذه الدورة تصميم الموقع من الألف إلي الياء بطريقة عملية 100% وباحدث التقنيات والاساليب


ما هي CSS؟

كلمة CSS هي إختصار لجملة Cascading Style Sheet و هي لا تعتبر لغة برمجة بل لغة توصيفية و تستخدم لعمل تنسيق لموقعك و جعل شكله رائع , تعد لغة ال CSS صديقة للغة ال HTML فتجد انه يتم ذكرهم مع بعض دائماً اذا تحدثتنا عن تصميم وإنشاء صفحات الويب , ف هاتين اللغتان هم اساس تصميم مواقع الويب فمن خلال HTML يتم تصميم هيكل صفحة الويب و بال CSS يتم تنسيق ذلك الهيكل اذا نفهم من ذلك ان CSS لغة مكملة ل HTML

هل لغة CSS كافية لعمل موقع كامل؟

بالطبع فان لغة CSS قوية ولا غنى عنها فبدونها لن تستطيع تنسيق صفحات الويب و ستكون صفحة الويب لديك بيضاء بلا شكل ولا روح لكن لغة css وحدها لا تكفي و كما ذكرنا ان هذه اللغة في الاساس دورها هو تنسيق html فلن تستطيع استخدام css في الويب بدون لغة html
