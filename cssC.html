Description


CSS هو اختصار لـ (Cascading Style Sheets) وهي لغة برمجة تستخدم لتنسيق وتصميم صفحات الويب. تستخدم CSS لجعل صفحات الويب أكثر جمالًا وتنظيمًا من خلال تحديد الألوان والخطوط والمسافات والأشكال والأبعاد والزوايا والتدرجات وغيرها من العناصر التي تؤثر على مظهر الصفحة.


بالإضافة إلى تحديد مظهر الصفحة، يمكن استخدام CSS لتنظيم المحتوى وتحسين تجربة المستخدم من خلال تحديد نمط العرض وطريقة التفاعل مع الصفحة


وتتكون أساسيات CSS من العناصر والخصائص والقواعد، حيث تحدد العناصر المستخدمة في الصفحة والخصائص المرتبطة بها والقواعد التي تحدد كيفية تطبيق الخصائص على العناصر.
